🧠 Orchestrator Key Behaviors

Pre-Flight Protocol

  • Scout code before asking questions
  • Max 2-3 targeted questions
  • Structured options, not open Q's
  • Three interruption tiers

Tool Suggestions

  • Identify project type automatically
  • Suggest tools that unlock autonomy
  • Playwright #1 for web projects
  • Match suggestions to task scope

Research Flagging

  • Research default-on, not opt-in
  • Domain complexity decision tree
  • Constraints flow to builder tasks
  • 5-min research saves 50min debugging
